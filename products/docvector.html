DocVector

Multi-format document ingestion pipeline for OpenSearch with automatic vectorization and RAG search.

Core Features

  • Multi-Format Ingestion: PDF, Word, Excel, Markdown, Code (Python, Java, JS), YAML
  • Automatic Vectorization: Embeddings via Sentence Transformers
  • OpenSearch k‑NN: Vector index with hybrid search (semantic + keyword)
  • Web UI: Upload, ingest and search through a simple GUI
  • RAG Search: Intelligent document retrieval on vectorized content

Deployment & Configuration

  • Docker Compose: Quick start with simple commands
  • Kubernetes/Helm: Production-ready with official chart
  • REST API: Programmatic interface for developers
  • Curl Commands: Complete documentation of endpoints

Technology Stack

  • Backend: Python 3.11+ with FastAPI
  • Database: OpenSearch 3.x for vectors
  • Orchestration: Docker & Kubernetes
  • Search Engine: FAISS (k‑NN Engine)

About

DocVector is a multi-format document ingestion pipeline for OpenSearch with automatic vectorization and RAG search. Perfect for system administrators and DevOps engineers.
