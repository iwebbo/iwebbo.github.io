  • Project-Based Organization: Isolate document collections and conversations by project
  • Real-Time Streaming: Server-Sent Events for progressive responses

Agent Library

Each agent is defined by a single .md file with YAML frontmatter. Register via POST /api/skills/register — credentials resolved securely from DB, never in plain config.

Open-source alternative to OpenClaw & Claude Computer Use

AgentRAG.io brings the same skill-based agent paradigm — self-hosted, fully controllable, multi-provider. Drop a .md file to create an agent. Connect SSH or WinRM hosts once, run anywhere. No vendor lock-in, no usage caps.

Linux SSH Agents SSH

nginx_ops

Manage Nginx — config test, reload, log analysis, upstream health, SSL expiry check.

"nginx is returning 502 on prod, check upstream and reload" "Show me the top 10 slowest endpoints from access.log" "Check SSL cert expiry on all vhosts"
apache_ops

Manage Apache — VHost management, mod_status, graceful restart, SSL, access log analysis.

"Test Apache config and do a graceful restart" "Enable mod_rewrite and reload" "Which IPs generated the most 403s today?"
kubernetes_node_ops

Inspect Kubernetes — pod triage, CrashLoopBackOff diagnosis, node drain, rolling restarts, rollback.

"List all pods in CrashLoopBackOff and show their last logs" "Drain node k8s-node-02 for maintenance" "Rollback the payment-service deployment"
docker_ops

Operate Docker — container lifecycle, image pull, health checks, Compose redeploy, prune.

"Pull latest image and recreate the api container" "Show memory usage for all running containers" "Prune unused images and volumes to free disk"
postgres_ops

Monitor PostgreSQL — replication lag, slow queries, lock contention, bloat, backup status.

"Check replication lag on all replicas" "Find the top 5 slowest queries this hour" "Which tables have the most dead tuples and need VACUUM?"
mysql_ops

Monitor MySQL/MariaDB — processlist, replica lag, slow log, InnoDB metrics, backup check.

"Show full processlist and kill queries running over 30s" "Check replica status on db-replica-01" "Top 10 slowest queries from slow log this week"
linux_incident

Triage Linux incidents — CPU/memory/disk pressure, OOM kills, systemd failures, network saturation.

"The server is slow, run a full triage and report" "Check for OOM kills in the last hour" "List all failed systemd units and their last error"
log_analyzer

Analyse any log — grep/awk/tail, error patterns, log rotation, SFTP download.

"Count error occurrences by type in /var/log/app.log" "Force log rotation and archive logs older than 7 days" "Download the last nginx error log for offline analysis"
deploy_rollback

Deploy & rollback — git pull, build, service restart, health check, auto-rollback on failure.

"Deploy main branch to prod-web-01 with health check" "Rollback to the previous commit on staging" "Run full deploy pipeline: pull → npm ci → reload service"

Windows WinRM Agents WinRM

iis_ops

Manage IIS — app pool lifecycle, site bindings, SSL certificates, access log analysis, FREB.

"Recycle the MyApp pool and check its state after" "Show all IIS sites with their bindings and SSL expiry" "Detect 500 errors in IIS logs from the last hour"
windows_incident

Triage Windows incidents — CPU/memory/disk pressure, event log errors, stuck processes, service failures.

"The server is unresponsive, run a full Windows triage" "List critical events from System log in the last 2h" "Find and kill the process consuming the most memory"
sqlserver_ops

Monitor SQL Server — blocking queries, AG replication lag, failed jobs, index fragmentation, backup status.

"Show all blocking sessions with their query text" "Check AG sync lag on all replicas" "List SQL Agent jobs that failed in the last 24h"
windows_deploy_rollback

Deploy & rollback — robocopy staging, app pool recycle, health check, auto-rollback, backup snapshots.

"Deploy staging build to IIS and verify health endpoint" "Rollback to yesterday's backup snapshot" "Stop pool → robocopy → start pool → health check pipeline"
windows_log_analyzer

Analyse Windows logs — Event Log queries, failed logins, lockouts, IIS W3C logs, custom app logs.

"Show all failed login attempts from the Security log today" "Which accounts got locked out in the last 24h?" "Search custom app log for EXCEPTION patterns"
